# Make beautiful, accessible websites.

Natura11y is an open-source design system for building modern websites. The framework aims to remove technical barriers for those with limited resources. It provides a wealth of customizable components accessible via CDN.

At the same time, the framework is ideal for seasoned developers and complex projects.
