## Files at a Glance

Compared to other front-end frameworks, Natura11y's source code is diminutive. **Less code equals less technical debt**. Figure 1 below depicts the overall folders and files that make up Natura11y proper:

</TextBlock>

<FigureBlock 
    figureNumber="1"
    figureWidth="narrow"
    codeLanguage="treeview"
    codeToolbar={false}
    highlightedLines="4,6,8,9-11"
    codeExample={(`
    natura11y/
    ├── dist/
    │   ├── css/
    │   │   └── natura11y.css
    │   ├── js/
    │   │   └── natura11y.js
    ├── src/
    │   └── modules/
    │       ├── [module-name]/
    │       │   ├── _style.scss
    │       │   └── index.js
    │       └── index.js
    ├── images/
    └── index.html
    `)}
/>

<TextBlock>

Notice the **/src/modules** <a href="https://github.com/cavidano/natura11y/tree/main/src/modules" target="_blank">directory</a> shown above in Figure 1, line 8. It contains Natura11y's *modules*. Each *module* (e.g., *accordion*) is a directory that contains two files: **_style.scss** and **index.js**. In this way, styles (CSS) and functionality (JS) reside together for each module.

Natura11y uses <a href="https://webpack.js.org/" target="_blank">Webpack</a> to bundle all modules into a single CSS file, and a single JavaScript file. Figure 1, above, highlights these files: **/dist/css/natura11y.css** on line 4, and **/dist/js/natura11y.js** on line 6.

These files are two of the three dependencies required to use Natura11y. The third dependency is for icons. Follow the *Quick Start* instructions below for including these dependencies (via CDN).

</TextBlock>

## Quick Start

Get up and running with Natura11y. Create a plain HTML document and add the code from Figure 2 (below).

</TextBlock>

<FigureBlock 
    figureNumber="2"
    figureWidth="medium"
    highlightedLines="3,13,16,31"
    codeExample={(`
    <!DOCTYPE html> \n
    <html dir="ltr" lang="en"> \n
        <head> \n
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"> \n
            <title>Page Title • Your Website Name</title> \n
            <!-- Natura11y Icons -->
            <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cavidano/natura11y-icons@main/dist/natura11y-icons.min.css"> \n
            <!-- Natura11y Stylesheet -->
            <link rel="stylesheet" href="${cdnCSS}"> \n
        </head> \n
        <body> \n
            <header id="global-header"></header> \n
            <main id="skip-header-content">
                <h1>Hello Nature</h1>
            </main> \n
            <footer id="global-footer"></footer> \n
            <!-- Natura11y Javascript -->
            <script src="${cdnJS}"></script> \n
        </body> \n
    </html>
    `)}
/>

<TableLineDescBlock targetFigure="2">

<TableLineDescRow lines="3">

On the HTML tag, the `lang="en"` attribute sets the default language (English). The `dir="ltr"` attribute sets the default text direction (left to right).

</TableLineDescRow>

<TableLineDescRow lines="13">

CDN Stylesheet link to <a href="/icons">Natura11y Icons</a>, our very own open-source icon library.

</TableLineDescRow>

<TableLineDescRow lines="16">

CDN link to the framework's latest stable version of the core stylesheet.

</TableLineDescRow>

<TableLineDescRow lines="31">

CDN link The framework's latest stable core Javascript file.

</TableLineDescRow>

</TableLineDescBlock>
