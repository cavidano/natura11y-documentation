## Framework at a Glance

Natura11y is built with modular CSS (SASS) and JavaScript (es6+). Figure 1 (bellow) depicts the general folder and file tree that makes up the framework proper:

</TextBlock>

<FigureBlock 
    figureNumber="1"
    figureWidth="narrow"
    codeLanguage="treeview"
    codeToolbar={false}
    highlightedLines="4,6,8"
    codeExample={(`
    natura11y/
    ├── dist/
    │   ├── css/
    │   │   └── natura11y.css
    │   ├── js/
    │   │   └── natura11y.js
    ├── src/
    │   └── modules/
    │       ├── [module-name]/
    │       │   ├── _style.scss
    │       │   └── index.js
    │       └── index.js
    ├── images/
    └── index.html
    `)}
/>

<TextBlock>

The **src/modules/** <a href="https://github.com/cavidano/natura11y/tree/main/src/modules" target="_blank">directory</a> (Figure 1, line 8) contains the framework *modules*. Each *module* (e.g., *button*) is a directory containing two files: **_style.scss** and **index.js**.

We use a module bundler to process all modules into a single CSS file, and a single JavaScript file. This helps maintain (and future-proof) the framework while accounting for out-of-date browsers. Figure 1 highlights these files: **/dist/css/natura11y.css** (line 4), **dist/js/natura11y.js**/ (line 6).

These files are two of the three required dependencies of the framework. The third dependency is for icons. Follow our *Quick Start* instructions for including these dependencies (via CDN).

</TextBlock>

## Quick Start

Get up and running with Natura11y. Create a plain HTML document and add the code from Figure 2 (below).

</TextBlock>

<FigureBlock 
    figureNumber="2"
    figureWidth="medium"
    highlightedLines="3,13,16,31"
    codeExample={(`
    <!DOCTYPE html> \n
        <html lang="en" dir="ltr"> \n
            <head> \n
                <meta charset="utf-8">
                <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"> \n
                <title>Page Title • Your Website Name</title> \n
                <!-- Natura11y Icons -->
                <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cavidano/natura11y-icons@main/dist/natura11y-icons.min.css"> \n
                <!-- Natura11y Stylesheet -->
                <link rel="stylesheet" href="${cdnCSS}"> \n
            </head> \n
            <body> \n
                <header id="global-header"></header> \n
                <main id="skip-header-content">
                    <h1>Hello Nature</h1>
                </main> \n
                <footer id="global-footer"></footer> \n
                <!-- Natura11y Javascript -->
                <script src="${cdnJS}"></script> \n
            </body> \n
        </html>
    `)}
/>

<TableLineDescBlock targetFigure="2">

<TableLineDescRow lines="3">

On the HTML tag, the `lang="en"` attribute sets the default language (English). The `dir="ltr"` attribute sets the default text direction (left to right).

</TableLineDescRow>

<TableLineDescRow lines="13">

CDN Stylesheet link to <a href="/icons">Natura11y Icons</a>, our very own open-source icon library.

</TableLineDescRow>

<TableLineDescRow lines="16">

CDN link to the framework's latest stable version of the core stylesheet.

</TableLineDescRow>

<TableLineDescRow lines="31">

CDN link The framework's latest stable core Javascript file.

</TableLineDescRow>

</TableLineDescBlock>
