<TextBlock>

## The Button Selector

Buttons within Natura11y require a `.button` utility class on an `<a>` or `<button>` tag, shown in Figure 1. By default, `.button` selectors inherit text color and font size from their parent.

The example below is a generic `.button` selector with no other styles yet applied.

</TextBlock>

<FigureBlock
    figureNumber="1"
    visualExample={(
        <div class="text-align-center">
            <a class="button" href="#1">
                Button
            </a>
        </div>
    )}
    codeExample={(`
        <a class="button" href="#1">
            Button
        </a> \n
        <button class="button">
            Button
        </button>
    `)}
/>

<TextBlock>

## Full-width Buttons

For buttons that span the width of their parent, add the .width-100 <a href="#1">size utility class</a> (Figure 3).

Do not use `.display-block` for full-width buttons. Buttons have a display property of flex (for vertical centering text with icons). Using use `.display-block` changes the buttons display property from `flex` to `block`. This breaks the vertical centering of the icon within the button (See <a href="#1">buttons with text and icon</a> below).
</TextBlock>

<FigureBlock
    figureNumber="3"
    visualExample={(
        <button class="button width-100 theme-primary">
            Button
        </button>
    )}
    codeExample={(`
        <a class="button width-100 theme-primary" href="#1">
            Button
        </a>
    `)}
/>


<Divider id="outlinedButtons" />

<TextBlock>

## Outlined Buttons

Create a button with an outline instead of a fill. Add the outline modifier class `.button--outline` to the `.button` selector. The outline width is a <strong>2x</strong> multiple of your theme's body border width.

The outline color inherits the button's text color. You can change the color using our <a href="#1">text color utility classes</a>.

</TextBlock>

<FigureBlock
    figureNumber="4"
    visualExample={(
        <div class="grid grid--column-2--md gap-3">            
            <a class="button button--outline display-block" href="#1">
                Button
            </a>
            <a class="button button--outline display-block text-color-link" href="#1">
                Button
            </a>
        </div>
    )}
    codeExample={(`
        <a class="button button--outline" href="#1">
            Button
        </a> \n
        <a class="button button--outline text-color-link" href="#1">
            Button
        </a>
    `)}
/>

<TextBlock>

## Button Border Radius

Globally, all <code>.button</code> selectors have a border radius. This value is set with the <code>:root</code> scoped CSS Variable <code>--button-border-radius</code> (see Figure 5). Learn how to <a href="{{ site.url }}/docs/customization/#referenceGuideCSSVariables">customize CSS Variables</a>. 

</TextBlock>

<FigureBlock
    figureNumber="5"
    codeLanguage="css"
    visualExample={(
        <a class="button width-100 theme-primary" style={{'--button-border-radius': '0'}} href="#1">
            Button
        </a>
    )}
    codeExample={(`
        /* Globally remove border radius in CSS */ \n
        :root {
            --button-border-radius: 0;
        }
    `)}
/>

<TextBlock>

## Sizing Buttons

With Natura11y, `.button` selectors inherit their font size. Padding is set with relative (em) sizes. This allows you to resize multiple buttons at once. Use our <a href="#1">font size</a> utility classes, or declare your own CSS values.

The example below applies the `.font-size-sm` utility class to a `<ul>` containing buttons. Each button within this `<ul>` becomes smaller with this single declaration.

</TextBlock>

<Divider id="textAndIconButtons" />

<TextBlock>

## Text and Icon Buttons

Add an icon to a `.button` selector with the `.button--has-icon` modifier class (Figure 7). Buttons with icons require two child selectors.

The first child is is a `&lt;span&gt;` with its combined <a href="{{ site.url }}/icons">icon</a> classes. The second child is a `&lt;span&gt;` with a `.button__text` class. This holds the button's text. Further, the `.button__text` selector is our target for underlining the text on `:hover` and `:focus` states. Further still, it provides a small margin to separate the text from the icon.

</TextBlock>

<FigureBlock
    figureNumber="8"
    visualExample={(
        <div class="grid grid--column-2--md gap-3">
            <a class="button button--has-icon width-100 theme-primary" href="#1">
                <span class="icon icon-edit"></span>
                <span class="button__text">Edit</span>
            </a>  
            <a class="button button--has-icon width-100 button--outline" href="#1">
                <span class="icon icon-delete"></span>
                <span class="button__text">Delete</span>
            </a>
        </div>
    )}
    codeExample={(`
        <a class="button button--has-icon theme-primary" href="#1">
            <span class="icon icon-edit"></span>
            <span class="button__text">Button</span>
        </a> \n
        <a class="button button--has-icon button--outline" href="#1">
            <span class="icon icon-delete"></span>
            <span class="button__text">Delete</span>
        </a>
    `)}
/>
