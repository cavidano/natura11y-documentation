## Article Layout

The *article* component includes `.article--column-{x}` selector on an `<article>` tag. Here ***x*** equals the amount of columns (2 or 3). 

With the example below `.article--column-2` creates a two-column layout.

</TextBlock>

<TextBlock>

Figure 1 (below) provides the minimum HTML for the primary navigation component.

</TextBlock>

    figureNumber="1"

            <div class="article__body">
                ...
            </div> \n
        </article>
    `)}
/>

<TableLineDescBlock targetFigure="1">

<TableLineDescRow lines="1">

The parent `.article--column-{x}` selector is required.

Here ***x*** equals the amount of columns (either 2 or 3).

</TableLineDescRow>

<TableLineDescRow lines="3-5">

The `.article__header` selector is optional. 

It holds the article's `header` tag.

</TableLineDescRow>

<TableLineDescRow lines="7-9">

The `.article__sidebar` selector is required.

It holds the article's navigation content.

</TableLineDescRow>

<TableLineDescRow lines="11-13">

The `.article__body` selector is required. 

It holds the article's main content.

</TableLineDescRow>

</TableLineDescBlock>
