## The Collapse Relationship

The collapse mechanism establishes a relationship between a button and its target selctor. With each button press, the target's visibility toggles between <em>show</em> and <em>hide</em>.

</TextBlock>

<TableLineDescBlock targetFigure="1">

<TableLineDescRow lines="3-4">

On a  `<button>` tag, the `data-toggle="collapse"` attribute is required.

The <code>data-target-toggle="#target-id"</code> attribute equals the value of its target selector's <code>id</code> attribute.

</TableLineDescRow>

<TableLineDescRow lines="9-10">

The <code>.collapse</code> class is required.

It initially hides the target selector.

The <code>id</code> attribute equals the value of its associated button's <code>data-target-toggle</code> attribute.

</TableLineDescRow>

</TableLineDescBlock>
