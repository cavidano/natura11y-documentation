Change the <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/display" target="_blank">display</a> property of an element.

<Divider id="display-utilities" />

<TextBlock>

## Display Utilities

Natura11y provides the following display utilities:

</TextBlock>

<FigureBlock 
    figureNumber="1"
    figureWidth="medium"
    codeExample={(`
        <div class="display-block">...</div>
        <div class="display-inline">...</div>
        <div class="display-inline-block">...</div>
        <div class="display-flex">...</div>
        <div class="display-inline-flex">...</div>
        <div class="display-grid">...</div>
        <div class="display-inline-grid">...</div>
        <div class="display-none">...</div>
    `)}
/>

<TextBlock>

### Breakpoint Modifiers

Display utilities accept <a href="/docs/breakpoints/#breakpointModifiers">breakpoint modifiers</a>.

</TextBlock>

<FigureBlock 
    figureNumber="2"
    figureWidth="medium"
