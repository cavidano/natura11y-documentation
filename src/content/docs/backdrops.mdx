<TextBlock>

Backdrops provide a useful design pattern—placing a translucent image behind contextual content. Backdrops offer two distinct style variants: <a href="#imageProportional">Image proportional</a> and <a href="#fixedHeight">fixed height</a>. 

</TextBlock>

<AccessibilitySpotlight>

Backdrops remove the need for CSS background images. Feature images exist within the HTML, so screen readers can read the text of the image's *alt* attribute.

</AccessibilitySpotlight>

## Image Proportional

Image proportional backdrops scale and stack responsively. Below is an example of a basic image proportional backdrop.

</TextBlock>

<FigureBlock
    figureNumber="1"
    visualExample={(
        <ImageProportional />

    codeExample={(`
    <div class="backdrop theme-dark border-radius"> \n
        <div class="backdrop__credit">
            <p>Photo by <a href="#1">Jane Doe</a> on Unsplash</p>
        </div> \n
        <div class="backdrop__image">
           ${codeImgPlaceholderBackdrop}
        </div> \n
        <div class="backdrop__cover"> \n
            <div class="container text-align-center">
                <p class="h4 text-shadow margin-bottom-3">
                    <em>In Their Eternal Faces<br>The Sun—with just delight</em>
                </p>
                <a class="button font-size-md theme-primary" href="#1">
                    Emily Dickenson Poetry
                </a>
            </div> \n
        </div> \n
    </div>
    `)}
/>

<TableLineDescBlock targetFigure="1">

<TableLineDescRow lines="1">

The parent `.backdrop` selector is required.

The `.theme-dark` utility class is optional. It provides the background and text color. If not present, these colors are inherited from the backdrop's nearest ancestor (with a theme color applied).

The `.border-radius` utility class is optional.

</TableLineDescRow>

<TableLineDescRow lines="3-5">

The `.backdrop__credit` selector is optional.

It is where you provide a credit for the image.

</TableLineDescRow>

<TableLineDescRow lines="7-9">

The `.backdrop__image` selector is required.

It contains the feature image.

</TableLineDescRow>

<TableLineDescRow lines="11-22">

The `.backdrop__cover` selector is required. 

It holds the backdrop's contextual content.

</TableLineDescRow>

</TableLineDescBlock>

<TextBlock>

The size of this backdrop (proportional width and height) is determined by the size of its feature image.

The translucent effect of the backdrop is acheived with the use of additional utility classes. Our <a href="#1">theme color utilities</a> (e.g. <code>.theme-dark</code>) provide the background and text color. The <a href="#1">opacity utility class</a> (e.g. <code>.opacity-30</code>) applied to the backdrop's feature image, allows the background color to show through.

</TextBlock>
