<TextBlock>

Backdrops provide a useful design pattern—placing a translucent image behind contextual content. Backdrops offer two distinct style variants: <a href="#imageProportional">Image proportional</a> and <a href="#fixedHeight">fixed height</a>. 

</TextBlock>

<AccessibilitySpotlight>

Backdrops remove the need for CSS background images. Feature images exist within the HTML, so screen readers can read the text of the image's *alt* attribute.

</AccessibilitySpotlight>
