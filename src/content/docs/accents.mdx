## Gradient Veneer

Apply a directional gradient to any selector using the `.gradient-veneer-{x}` utilities. Where ***x*** specifies the gradient's direction: <em>top</em>, <em>right</em>, <em>bottom</em>, or <em>left</em> (Figure 1).
	
</TextBlock>

<FigureBlock
    figureNumber="1"
    codeExample={(`
        <div class="gradient-veneer-top">...</div>
        <div class="gradient-veneer-right">...</div>
        <div class="gradient-veneer-bottom">...</div>
        <div class="gradient-veneer-left">...</div>
    `)}
/>
