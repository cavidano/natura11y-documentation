## Gradient Veneer

Apply a directional gradient to any selector using the `.gradient-veneer-{x}` utilities. Where ***x*** specifies the gradient's direction: *top*, *right*, *bottom*, or *left* (Figure 1).
	
</TextBlock>

<FigureBlock
    figureNumber="1"
    visualExample={
        <GradientVeneerDirections />
    }
    codeExample={(`
        <div class="gradient-veneer-top">...</div>
        <div class="gradient-veneer-right">...</div>
        <div class="gradient-veneer-bottom">...</div>
        <div class="gradient-veneer-left">...</div>
    `)}
/>

<TextBlock>

Gradient veneers use a single color, fully transparent to fully opaque. They inherit their color from the nearest ancestrial background color (usually applied with <a href="#1">theme color utilities</a>).

</TextBlock>

<FigureBlock
    figureNumber="2"
    visualExample={(
        <GradientVeneerThemes />
    )}
    codeExample={(`
        <div class="backdrop theme-primary"> \n
            <div class="backdrop__image gradient-veneer-bottom">
                <!-- <img src="./path/to/image.jpg" alt="Placeholder"> -->
            </div> \n
            <div class="backdrop__cover">
                ...
            </div> \n
        </div>
    `)}
    
/>

<TextBlock>

Gradient veneer utilities place the gradient (as a psuedo element) above all child selectors on the <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/z-index" target="_blank">z index</a>. With the above examples, we applied a gradient veneer utility to the `.backdrop-image` selector. This allows the gradient to appear above its child `<img>` tag (Figure 2, lines 3-5).

</TextBlock>

## Subtle Fill

The `.subtle-fill` utility class adds a light tint background to any selector. Subtle fills inherit their color from the **text color** of the element on which they are placed.

With the card examples below, we can add soft emphasis to the action buttons within the `.card__footer` selector.

Notice the subtle fill appears darker within the *white theme* card, while appearing lighter on the *dark theme* card.

</TextBlock>

<FigureBlock
    figureNumber="3"
    visualExample={(
        <SubtleFill />
    )}
    highlightedLines="7"
    codeExample={(`

        <div class="card__body">
            ...
        </div> \n

            ...
        </div> \n
    </div>
    `)}
