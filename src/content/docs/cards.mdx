## The Card Component

The example below provides a rudimentary card component.

</TextBlock>

<FigureBlock
    figureNumber="1"
    highlightedLines="1,3-5,7-14,16-29"
    visualExample={(
        <CardExample />

    codeExample={(`
        <div class="card border"> \n
            <div class="card__head border-bottom">
                <p>Tropical flowers</p>
            </div> \n
            <div class="card__body">
                <h2 class="h4">
                    Gustavia superba
                </h2>
                <p>
                    Gustavia superba is an understory tree that grows in <a href="#1">Central</a> and north-western <a href="#1">South America</a>. Common names include membrillo, sachamango and heaven lotus.
                </p>
            </div> \n
            <div class="card__foot border-top font-size-sm">
                <ul class="nav nav--horizontal justify-content-between">
                    <li>
                        <a href="#1">
                            Secondary Action
                        </a>
                    </li>
                    <li>
                        <a class="button button--outline" href="#1">
                            Primary Action
                        </a>
                    </li>
                </ul>
            </div> \n
        </div>
    `)}
/>

<TableLineDescBlock targetFigure="1">

<TableLineDescRow lines="1">

The parent `.card` selector is required.

The `.border` utility class not required.

It provides the card's outer border.

</TableLineDescRow>

<TableLineDescRow lines="3-5">

The `.card__head` selector is not required.

It holds the card's small title.

The `.border-bottom` utility class is not required.

It provides the card header's bottom border.

</TableLineDescRow>

<TableLineDescRow lines="7-14">

The `.card__body` selector is required.

It holds the card's main message. Cards can have multiple `.card__body` selectors.

</TableLineDescRow>

<TableLineDescRow lines="16-29">

The `.card__foot` selector is not required.

It selector holds the card's actionable buttons and links.

The `.border-top` utility class is not required.

It provides the card footer's top border. 

</TableLineDescRow>

</TableLineDescBlock>

## Including Images

Include an image within a card. Add the `<img>` as a sibling of `.card__body`. Images do not need a class.

</TextBlock>
