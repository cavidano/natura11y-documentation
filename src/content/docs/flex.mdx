<TextBlock>

Natura11y's <a href="https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox" target="_blank">flexbox</a> utilities provide options for creating one-dimensional layouts. Some flexbox utilities deal with horizontal and vertical alignment. These utilities work alongside the <a href="#1">navigation</a>, <a href="#1">grid</a>, and in some cases, <a href="/docs/backdrops">backdrop</a> components.

</TextBlock>

<Divider id="flexDirectionUtilities" />

<TextBlock>

## Flex Direction Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction">flex direction</a> utilities (Figure 1) create the flex container. They define the main axis: *row* or *column*, along with direction: *normal* or *reverse*. Further, they set the <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-wrap">flex-wrap</a> property to *wrap*.

</TextBlock>

<FigureBlock
    figureNumber="1"
    visualExample={(
        <Direction />
    )}
    codeExample={(`
        <div class="flex-row">...</div>
        <div class="flex-row-reverse">...</div>
        <div class="flex-column">...</div>
        <div class="flex-column-reverse">...</div>
    `)}
/>

<TextBlock>

Need a flex row that doesn't wrap? Replace `.flex-row` with `.display-flex` (from the <a href="/docs/display/">display utilities</a>).

</TextBlock>

<TextBlock>

### Breakpoint Modifiers

Flex direction utilities accept <a href="/docs/breakpoints/#breakpointModifiers">breakpoint modifiers</a>.

With the HTML in Figure 2, the flex container starts with the *column* direction. It then changes to the *row* direction at the large breakpoint.

</TextBlock>

<FigureBlock
    figureNumber="2"
    visualExample={(
        <DirectionBreakpointModifier />

    codeExample={(`
        <!-- With breakpoint modifiers --> \n

<TextBlock>

## Flex Grow Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-grow" target="_blank">Flex grow</a> utilities set the grow factor of a flex item along the flex container's main axis. The default grow value of a flex item is *0*.

</TextBlock>

<FigureBlock
    figureNumber="3"
    codeExample={(`
        <div class="flex-grow-1">...</div>
        <div class="flex-grow-0">...</div>
    `)}
/>

<TextBlock>

With `.flex-grow-1`, applied to a flex item, it will grow to fill any available space.

</TextBlock>

<FigureBlock
    figureNumber="4"
    visualExample={(
        <Grow />
    )}
    codeExample={(`
        <div class="display-flex subtle-fill">
            <div class="flex-grow-1 padding-x-3 padding-y-2 theme-white border">
                <p>Flex Item</p>
            </div>
            <div class="padding-x-3 padding-y-2 theme-white border">
                <p>Flex Item</p>
            </div>
        </div>
    `)}
/>

<Divider id="flexShrinkUtilities" />

<TextBlock>

## Flex Shrink Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-shrink" target="_blank">Flex shrink</a> utilities (e.g., `.flex-shrink-{x}`) set the shrink factor of a flex item, where <strong>*x*</strong> equals *0*, *1*, or *2*.These three decimals represent the amount which the flex item shrinks.

</TextBlock>

<FigureBlock
    figureNumber="5"
    codeExample={(`
        <div class="flex-shrink-0">...</div>
        <div class="flex-shrink-1">...</div>
        <div class="flex-shrink-2">...</div>
    `)}
/>

<TextBlock>

With a flex shrink value of *0*, a flex item will not shrink as it's flex container gets smaller. With a shrink value of *1*, the flex item will shrink, but no smaller than it’s content. Finally, a flex item with a flex shrink value of *2* will shrink smaller than it’s content.

With the examples below, `.flex-shrink-1` applied to the first flex item allows it to shrink along it's main axis.

</TextBlock>

<FigureBlock
    figureNumber="6"
    visualExample={(
        <Shrink />
    )}
    codeExample={(`
        <div class="display-flex">
            <div class="flex-shrink-1 padding-x-3 padding-y-2 theme-white border">
                <p>Flex Shrink</p>
            </div>
            <div class="padding-x-3 width-100 padding-y-2 theme-white border">
                <p>2</p>
            </div>
        </div>
    `)}
/>

## Justify Content Utilites

The <a href="#1">justify content</a> utilites (Figure 3) align items along the main-axis of their flex container.  

</TextBlock>

    visualExample={(
