<TextBlock>

Natura11y's <a href="https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox" target="_blank">flexbox</a> utilities provide options for creating one-dimensional layouts. Some flexbox utilities deal with horizontal and vertical alignment. These utilities work alongside the <a href="#1">navigation</a>, <a href="#1">grid</a>, and in some cases, <a href="#1">backdrop</a> components.

</TextBlock>

<Divider id="flexDirectionUtilities" />

<TextBlock>

## Flex Direction Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction">flex direction</a> utilities (Figure 1) create the flex container. They define the main axis: <em>row</em> or <em>column</em>, along with direction: <em>normal</em> or <em>reverse</em>. Further, they set the <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-wrap">flex-wrap</a> property to <em>wrap</em>.

</TextBlock>

## Flex Grow Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-grow" target="_blank">Flex grow</a> utilities set the grow factor of a flex item along the flex container's main axis. The default grow value of a flex item is <em>0</em>.

</TextBlock>

<TextBlock>
    ## Flex Shrink Utilities
    <p>
        <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-shrink" target="_blank">Flex skrink</a> utilities (e.g., <code>.flex-grow-{x}</code>) set the shrink factor of a flex item, where <strong><em>x</em></strong> equals <em>0</em>, <em>1</em>, or <em>2</em>.These three decimals represent the amount which the flex item shrinks.
    </p>
</TextBlock>

<figure class="container medium margin-y-4">
    <pre>
        <code class="language-html">
            &lt;div class="flex-shrink-0"&gt;...&lt;/div&gt;
            &lt;div class="flex-shrink-1"&gt;...&lt;/div&gt;
            &lt;div class="flex-shrink-2"&gt;...&lt;/div&gt;
        </code>
    </pre>
    <figcaption>
        Figure 6
    </figcaption>
</figure>

<TextBlock>
    <p>
        With a flex shrink value of <em>0</em>, a flex item will not shrink as it's flex container gets smaller. With a shrink value of <em>1</em>, the flex item will shrink, but no smaller than it’s content. Finally, a flex item with a flex shrink value of <em>2</em> will shrink smaller than it’s content.
    </p>
    <p>
        With the examples below, <code>.flex-shrink-1</code> applied to the first flex item allows it to shrink along it's main axis. 
    </p>
</TextBlock>
