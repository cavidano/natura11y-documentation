<TextBlock>

Natura11y's <a href="https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox" target="_blank">flexbox</a> utilities provide options for creating one-dimensional layouts. Some flexbox utilities deal with horizontal and vertical alignment. These utilities work alongside the <a href="#1">navigation</a>, <a href="#1">grid</a>, and in some cases, <a href="#1">backdrop</a> components.

</TextBlock>

<Divider id="flexDirectionUtilities" />

<TextBlock>

## Flex Direction Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction">flex direction</a> utilities (Figure 1) create the flex container. They define the main axis: *row* or *column*, along with direction: *normal* or *reverse*. Further, they set the <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-wrap">flex-wrap</a> property to *wrap*.

</TextBlock>

{/* figure */}

<TextBlock>

Need a flex row that doesn't wrap? Replace `.flex-row` with `.display-flex` (from the <a href="/docs/display/">display utilities</a>).

</TextBlock>

## Flex Grow Utilities

The <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-grow" target="_blank">Flex grow</a> utilities set the grow factor of a flex item along the flex container's main axis. The default grow value of a flex item is *0*.

</TextBlock>

<TextBlock>
    ## Flex Shrink Utilities
    <p>
        <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-shrink" target="_blank">Flex skrink</a> utilities (e.g., `.flex-grow-{x}`) set the shrink factor of a flex item, where <strong>*x*</strong> equals *0*, *1*, or *2*.These three decimals represent the amount which the flex item shrinks.
    </p>
</TextBlock>

{/* figure */}

<TextBlock>
    <p>
        With a flex shrink value of *0*, a flex item will not shrink as it's flex container gets smaller. With a shrink value of *1*, the flex item will shrink, but no smaller than it’s content. Finally, a flex item with a flex shrink value of *2* will shrink smaller than it’s content.
    </p>
    <p>
        With the examples below, `.flex-shrink-1` applied to the first flex item allows it to shrink along it's main axis. 
    </p>
</TextBlock>
